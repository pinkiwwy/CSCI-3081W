Detailed Description

A representation of a distress sensor. Sensor has a range that sense robots nearby, but it has no sense about the coming direction. For our purposes, the output will be 1 if distress call exists, and 0 if no robots nearby.

int SensorDistress::Output ( void  )

Compute calculation of distance range between robots based on the distress event.

getter and setter for position and radius. In SensorDistress, the sensor will sense if the distance between robots is in range. If so, it is a distress call. The call will not be determining the angle of direction. getter for radius, used for finding the range. Maybe having 2*radius as the range for sensing. getter and setter for position, use for calculate distance between robots. No direction in distress class, so no heading angle. Note that proximity events will return 1 if there is in range distress call. Or it will 0 as none.
